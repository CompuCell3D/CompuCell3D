cc3d.core.Validation.sanity_checkers module¶

cc3d.core.Validation.sanity_checkers.validate_cc3d_entity_identifier(entity_identifier: str, entity_type_label: str = '')¶

Checks if entity identifier conforms to CC3D naming standards. Cell types names, fields etc have to start from the letter and contain only alphanumeric strings with no spaces or charcters that could be interpreted as syntax elements in python . Function does not return anything but raises AttributeError exception if it detects inappropriate identifier :param entity_identifier: identifier to check :param entity_type_label: optional string that helps print out more-targeted error message :return:
