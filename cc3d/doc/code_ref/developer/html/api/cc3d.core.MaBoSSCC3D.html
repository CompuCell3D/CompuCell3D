cc3d.core.MaBoSSCC3D.maboss_model(bnd_file: Optional[str] = None, bnd_str: Optional[str] = None, cfg_file: Optional[str] = None, cfg_str: Optional[str] = None, time_step: float = 1.0, time_tick: float = 1.0, discrete_time: bool = False, seed: Optional[int] = None) → object¶

Instantiate a MaBoSS simulation instance from files and/or strings.

Parameters

  • bnd_file (str) – path to a network file (required unless bnd_str is specified)

  • bnd_str (str) – network multiline string (required unless bnd_file is specified)

  • cfg_file (str) – path to a configuration file (required unless cfg_str is specified)

  • cfg_str (str) – configuration multiline string (required unless cfg_file is specified)

  • time_step (float) – period of one simulation step (default 1.0)

  • time_tick (float) – period of one evaluation (default 1.0)

  • discrete_time (bool) – flag to use discrete time (default False)

  • seed (int) – random generator seed (default 0)

Returns

MaBoSS simulation instance

Return type

MaBoSSCC3DPy.CC3DMaBoSSEngine
