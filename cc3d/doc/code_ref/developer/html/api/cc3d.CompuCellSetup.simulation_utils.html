cc3d.CompuCellSetup.simulation_utils.extract_type_id_type_name_dict(cell_types_elements)¶

Extract dictionary mapping cell type id to cell type name from a sequence of xml elements that look as follows: <CellType TypeId=”0” TypeName=”Medium”/> <CellType TypeId=”1” TypeName=”Condensing”/> <CellType TypeId=”2” TypeName=”NonCondensing”/> Note that this sequence of elements can be found int he CellTypePlugin or in the <CompuCell3DLatticeData> in the dml.files :param cell_types_elements: :return:
